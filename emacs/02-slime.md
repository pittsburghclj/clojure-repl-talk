# Swank = SLIME server

* Leiningen plugin

* `lein swank` - run the server

* `M-x clojure-jack-in`

# Really small intro

    C-c C-z    go from Clojure buffer to REPL
    C-c M-p    switch namespace to current buffer
    C-c C-l    reload current file and namespaces

    (swank.core/break)    breakpoint in code
